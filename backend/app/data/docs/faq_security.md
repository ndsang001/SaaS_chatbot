# Security and Data Protection

## How is my data protected?

We use encryption in transit and at rest.
All communication is secured using HTTPS.

## Do you store passwords in plain text?

No. Passwords are hashed using industry-standard cryptographic hashing algorithms.

## How can I enable two-factor authentication?

You can enable two-factor authentication (2FA) from your Security Settings page.
Follow the instructions to connect your authenticator app.
